A Pomodoro timer application built in **C**, running entirely in the terminal. Developed as part of the UCD CyberForce challenge — El Jadida.

---

## 📖 What is the Pomodoro Technique?

The Pomodoro Technique is a simple and effective time management method:

- 🧠 **25 minutes** of focused work → this is one **Pomodoro**
- ☕ **5-minute short break** after each Pomodoro
- 🛋️ **30-minute long break** after **4 consecutive Pomodoros**
- 🔁 Repeat the cycle until the session ends

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Dostofine/MINUTEUR_POMODORO.git
cd MINUTEUR_POMODORO
```

### 2. Build
```bash
make
```

### 3. Run
```bash
make run 
or
./build/DOSTO_POMODORO
```

### 4. Clean build files
```bash
make clean
```
